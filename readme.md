# Monument Generator

AI-powered monument image generator using Google Gemini.

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env.local` file with your API key: `GEMINI_API_KEY=AIzaSyBmQs1pCoJZme9tMZyCdaOzGM-qQ901_ZM`
4. Run development server: `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Add `GEMINI_API_KEY` environment variable in Vercel dashboard
4. Deploy!
